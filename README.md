# ED_Lista03.03
## Inversão de Caracteres: Recursiva

Este desafio consiste em criar uma aplicação, em Java, que receba uma cadeia de caracteres e apresente sua inversão.
```
Requisitos do projeto:

1. Arquitetura MVC;
2. A inversão deve ser realizada por uma função recursiva;
3. Deve-se utilizar o comando .substring(posInicial,posFinal);
3. Declarar, via comentário, a condição de parada da função.
```
